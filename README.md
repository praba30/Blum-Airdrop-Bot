## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/your-username/Blum-Airdrop-Bot.git
   ```

2. **Install Dependencies:**
   ```bash
   cd Blum-Airdrop-Bot
   npm install
   ```

3. **Configuration:**
   - Update the `config.js` file with your Blum platform credentials.

4. **Run the Bot:**
   ```bash
   npm start
   ```
